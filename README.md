# FlutterNcpApp
***
本应用采用Dart语言编写查看NCP(新型冠状肺炎)疫情数据，采用Flutter跨平台技术，应用到封装网络请求，自定义组件(loading)，自定义绘制组件(时间轴)，两种跳转路由方式
